# Bro Code

Share text and files between devices using a 3-digit code.

## How (Send)
1. Enter text or select a file.
2. Click Send.
3. Use the 3-digit code generated.

## How (Receive)
1. Enter the 3-digit code.
2. Text will appear or file will download.

## Notes
- Items expire after 15 minutes.
- File limit is 2MB. ( Work in progress , Temporarily Disabled )

## Tech Stack
- Frontend - HTML / JS  / Tailwind CSS CDN
- Hosting - Github Pages
- Backened - Serverless
- Database - Supabase
